========================================
ШАГ 1: Установка PHP расширений (ТРЕБУЕТСЯ SUDO)
========================================

Для работы проекта необходимо установить PHP расширения:

    sudo bash install-php-extensions.sh

Этот скрипт установит все необходимые расширения:
- sqlite3 и pdo_sqlite (для базы данных)
- xml и dom (для обработки XML/DOM)
- curl, mbstring, gd, zip, bcmath, intl (для Laravel)
- mysql (на случай, если захотите использовать MySQL)

========================================
ШАГ 2: Запуск миграций базы данных
========================================

После установки расширений:

    cd /home/lime/Документы/nrchd.kz/nrchd.kz
    php artisan migrate

========================================
ШАГ 3: Запуск проекта
========================================

Для полного запуска проекта (Laravel + Vite):

    composer run dev

Этот скрипт запустит одновременно:
- Laravel сервер на http://127.0.0.1:8000
- Vite dev server на http://127.0.0.1:5173
- Очереди (queue:listen)
- Логи (pail)

Или можно запустить отдельно:

    # Терминал 1: Laravel сервер
    php artisan serve

    # Терминал 2: Vite dev server
    npm run dev
