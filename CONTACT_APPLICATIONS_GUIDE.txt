🎯 КАК ПОЛЬЗОВАТЬСЯ АДМИНКОЙ:
═══════════════════════════════════════════════════════════════

1. ПРОСМОТР СТАТИСТИКИ:
   • На главной странице видны карточки с количеством заявок по каждому направлению
   • Клик по карточке автоматически фильтрует заявки по этому направлению
   • Цветная левая граница у каждой заявки показывает её направление

2. ФИЛЬТРАЦИЯ:
   • Поиск: по имени, email, телефону, тексту сообщения
   • Направление/Форма: выбор конкретного направления
   • Статус: новые, в работе, решены, отклонены
   • Ответственный: назначенный пользователь

3. РАБОТА С ЗАЯВКОЙ:
   • Кликните на заявку для просмотра полной информации
   • Смените статус одним кликом
   • Добавьте заметки для коллег
   • Назначьте ответственного
   • Скачайте прикрепленные файлы

4. СТАТУСЫ ЗАЯВОК:
   🔵 Новая (new) - только что поступила
   🟡 В работе (in_progress) - ведется обработка
   🟢 Решена (resolved) - вопрос решен
   🔴 Отклонена (rejected) - заявка отклонена
