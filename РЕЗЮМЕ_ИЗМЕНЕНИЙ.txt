КАК ПРИМЕНИТЬ ИЗМЕНЕНИЯ:
========================

Шаг 1: Соберите фронтенд
-------------------------
npm run build

(ИЛИ для режима разработки: npm run dev)

Шаг 2: Очистите кэш Laravel
----------------------------
php artisan cache:clear
php artisan config:clear
php artisan view:clear

Шаг 3: Перезагрузите страницу в браузере
-----------------------------------------
Ctrl + F5 (принудительная перезагрузка)

ПРОВЕРКА РАБОТЫ:
===============

1. Откройте сайт - должен отобразиться на КАЗАХСКОМ языке
2. Проверьте header - все тексты должны быть на казахском
3. Нажмите кнопку переключения языка (RU) - сайт перезагрузится на русском
4. Нажмите кнопку переключения языка (EN) - сайт перезагрузится на английском
5. Нажмите кнопку (KZ) - вернетесь к казахскому
