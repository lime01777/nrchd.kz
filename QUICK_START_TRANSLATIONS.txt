ДОРАБОТКИ В АДМИНКЕ ПЕРЕВОДОВ:

✅ Проверка переводов в БД перед вызовом Google Translate
✅ Автоматическое переиспользование существующих переводов
✅ Детальная статистика (из БД / через Google)
✅ Оптимизация запросов к API
✅ Новый метод для перевода отдельных текстов

───────────────────────────────────────────────────────────────────
  📋 НАСТРОЙКА
───────────────────────────────────────────────────────────────────

1. Добавьте в файл .env:

   GOOGLE_TRANSLATE_API_KEY=ваш_ключ_здесь

2. Получите ключ на: https://console.cloud.google.com
   - Включите "Cloud Translation API"
   - Создайте API Key в разделе Credentials

3. Очистите кеш:

   php artisan config:clear
   php artisan cache:clear

───────────────────────────────────────────────────────────────────
  🚀 ИСПОЛЬЗОВАНИЕ
───────────────────────────────────────────────────────────────────

1. Откройте админку: http://127.0.0.1/admin/translations

2. Выберите целевой язык (Қазақша или English)

3. Нажмите "Запустить массовый перевод"

4. Наблюдайте за статистикой:
   - Сколько переводов взято из БД (экономия!)
   - Сколько новых переводов создано через Google
   - Общий прогресс

───────────────────────────────────────────────────────────────────
  📡 API ЭНДПОИНТЫ
───────────────────────────────────────────────────────────────────

Массовый перевод:
POST /admin/translations/translate-all
{
  "target_language": "kk"
}

Одиночный перевод:
POST /admin/translations/translate-single
{
  "text": "Текст",
  "target_language": "kk"
}
